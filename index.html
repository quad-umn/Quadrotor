Introduction

The purpose of this project is to set up a work environment that allows an autonomous control of a quadcopter (drone) and produce a set of documentation such that future staff or students could build on the work and/or improve on it

This project is designed to be the foundation of a course, or courses, that would be taught to students at the University of Minnesota and cover subjects including autonomous movement, 3D object tracking, and more
